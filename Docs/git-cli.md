# GIT CLI to Perform GIT Actions

-   To checkout the new branch in local

    ```Shell
     git checkout -b <branch-name>
    ```

-   To checkout the exsisting branch in local

    ```Shell
     git checkout b <branch-name>
    ```

-   To fetch from the remote branch

    ```Shell
     git fetch
    ```

-   To commit

    ```Shell
     git commit -m <commit-message>
    ```
